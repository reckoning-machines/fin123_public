Governance follows the work.

A Workbook view can analyze, explain, or discuss. It does not replace the governance chain underneath.

Model -> Version -> Scenario -> Run -> Results -> Replay -> Audit

  • Results are the typed output authority.
  • Audit is the evidence authority.
  • Replay reconstructs what happened.
  • Run Activity shows execution status.

Bring Your Model

Analysts should not have to rebuild working Excel models to enter the fin123 ecosystem.

  • Import existing Excel models instead of rebuilding them.
  • Preserve formulas, Bloomberg links, Visible Alpha links, and workbook structure.
  • Map live model inputs into approved =DATA() and governed execution paths.
  • Import comments into YAP chat for later promotion to Model Memory if approved.
  • Add versioning, Sheet Warnings, YAP context, Audit, replay, and =AI() without breaking analyst workflow.

YAP and Model Memory

YAP is in-workbook team chat. Approved discussion can become Model Memory, and Model Memory can inform future model runs.
